GLOSSARIO

Socket - Un socket è un punto finale di una connessione di rete tra un client e un server. È utilizzato per inviare e ricevere dati tra i dispositivi. Protocollo - Un insieme di regole che determinano come i dispositivi si comunicano in una rete. I protocolli comuni includono TCP e UDP. OSI - Modello a sette strati che definisce i livelli di comunicazione in una rete, dal livello fisico al livello applicazione. Client - Dispositivo che invia richieste a un server per accedere a risorse o servizi attraverso la rete. Server - Un sistema che fornisce risorse, servizi o dati a un altro sistema (client) attraverso una rete. TCP - Transmission Control Protocol, un protocollo di trasporto orientato alla connessione che garantisce la consegna affidabile dei dati. UDP - User Datagram Protocol, un protocollo di trasporto senza connessione che non garantisce la consegna dei dati.
